# LegacyBridge Social Media Content Optimizer

A comprehensive tool for creating platform-optimized social media content with AI-powered suggestions, generational translation, and multi-platform formatting.

## Features

- Text Editor & Formatter: Real-time preview for any social media platform
- Generational Translator: Adapt content for Gen Z, Millennial, Gen X, Boomer, Corporate, Developer audiences
- Multi-Platform Generator: Generate optimized versions for all platforms at once
- Leo AI: AI-powered content optimization suggestions
- Rich Formatting: Bold, italic, underline, strike-through, code blocks, quotes, callouts

## Platforms

Twitter/X, Facebook, LinkedIn, Instagram, TikTok, Reddit, Threads, Discord, Slack

## Generational Styles

Gen Z, Millennial, Gen X, Boomer, Corporate, Developer

## Installation

npm install
npm run dev
